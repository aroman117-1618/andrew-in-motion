# Andrew in Motion

**andrewinmotion.tech** is my professional portfolio website — designed and built with [Next.js](https://nextjs.org/), [React](https://react.dev/), and [Tailwind CSS](https://tailwindcss.com/). It highlights my expertise in GTM leadership, systems automation, and customer success, with interactive components and lightweight animations for a modern, responsive experience.

---

## 🚀 Features

- **Next.js App Router** for fast, modern page rendering  
- **Responsive design** optimized for desktop & mobile  
- **Interactive components** including flip cards and service toggles  
- **Glass & nebula backgrounds** built with custom React components  
- **Video demonstrations** embedded via `.webm` assets  
- **Optimized deployment** via Netlify  

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/)  
- **UI:** [React](https://react.dev/), [Tailwind CSS](https://tailwindcss.com/)  
- **Animations:** Framer Motion  
- **Deployment:** [Netlify](https://www.netlify.com/)  
- **Linting/Formatting:** ESLint & Prettier

## ⚡ Getting Started

### 1. Clone the repo
```bash git clone https://github.com/aroman117-1618/andrew-in-motion.git cd andrew-in-motion```

2. Install dependencies

```npm install```

3. Run locally

```npm run dev```

4. Build for production

```npm run build```
```npm start```


⸻

🌐 Deployment

This site is deployed via Netlify. Pushes to the main branch trigger automatic builds.
You can also run a production build locally for testing:

```npm run build && npm run start```
